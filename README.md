# PAROLA 🇮🇹

Un clone italiano di Wordle - indovina la parola di 5 lettere in 6 tentativi!

## 🎮 Gioca Ora

**[https://eponalab.github.io/parola/](https://eponalab.github.io/parola/)**

## 📖 Come Giocare

- Indovina la PAROLA in 6 tentativi
- Ogni tentativo deve essere una parola valida di 5 lettere
- Dopo ogni tentativo, il colore delle caselle cambierà:
  - 🟩 **Verde**: Lettera corretta nella posizione corretta
  - 🟨 **Giallo**: Lettera presente ma nella posizione sbagliata
  - ⬜ **Grigio**: Lettera non presente nella parola

## ✨ Caratteristiche

- 🗓️ **Nuova parola ogni giorno** - La parola cambia quotidianamente
- 📊 **Statistiche** - Traccia le tue vittorie, serie consecutive e distribuzione
- 📱 **Responsive** - Gioca su desktop, tablet o smartphone
- 🎨 **Design elegante** - Interfaccia pulita con gradiente viola
- 💾 **Salvataggio automatico** - Il tuo progresso viene salvato localmente
- 📋 **Condivisione** - Condividi i tuoi risultati con emoji (come Wordle!)
- ⌨️ **Supporto tastiera** - Usa sia la tastiera virtuale che quella fisica

## 🛠️ Tecnologie

- HTML5
- CSS3 (con animazioni e gradiente)
- JavaScript vanilla (nessuna dipendenza)
- LocalStorage per il salvataggio dello stato

## 📝 Lista Parole

- Oltre 150 parole italiane comuni per le soluzioni quotidiane
- Database esteso di parole valide per i tentativi
- Parole di 5 lettere selezionate per essere accessibili e interessanti

## 🎯 Esempi

```
PAROLA 4/6

⬜🟨⬜⬜⬜
⬜⬜🟩⬜⬜
🟩🟩🟩⬜⬜
🟩🟩🟩🟩🟩
```
